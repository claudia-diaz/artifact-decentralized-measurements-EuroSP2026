# artifact-decentralized-measurements-EuroSP2026
This is the Python simulation artifact that accompanies the paper "Decentralized Reliability Estimation for Low-Latency Mixnets", by Diaz, Halpin, and Kiayias, published at IEEE EuroS&amp;P 2026. 

The file Configuration_cd.py contains all the simulation variables that can be configured. After these have been set, main_cd.py can be run to generate simulation outputs. Those outputs are stored in files and read by the Jupyler scripts to generate figures. 

The simulation data files generated for the paper are unfortunately too big to add here.
